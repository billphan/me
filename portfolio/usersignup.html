User-Signup Interface

User-Signup is a web application interface. Developed using the Python based Flask framework and implements Flask templates and client-side validation.

Abstract

User-Signup is a form where users provide a username, password, and, optionally, an email address. There is an additional field to verify the password (the user must retype it exactly). After validation the input and either redirect them to a welcome page, or provide error information if any input field is invalid.

User-Signup is an exercise in web applications programming using Python, Flask, HTML, and CSS.
